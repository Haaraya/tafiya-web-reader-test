How Haaraya works

A reading world, not just a book library.

This test build proves the core reading loop: a child opens a book, reads inside the Tafiya reader, and moves through pages pulled from Supabase.

1

Choose a book

Children begin from a simple library card linked to a real book code.

2

Open the reader

The reader loads the book package from Supabase using the book code.

3

Read page by page

Each page displays the illustration and text in the approved Tafiya layout.

4

Build the journey

Later, passports, stamps, and progress will connect to this same reading loop.
